# 🎮 Game Library Manager v1.0

A console-based Game Library Manager built with Python using Object-Oriented Programming (OOP). This project allows users to manage their personal game collection by adding, viewing, searching, updating, deleting, saving, and loading games.

This project was created as my Intermediate Python capstone to practice software design, file handling, exception handling, and object-oriented programming.

---

## ✨ Features

- ➕ Add new games
- 📚 View all games
- 🔍 Search games by title (case-insensitive)
- ✏️ Update game information
- 🗑️ Delete games
- 💾 Save library to a file
- 📂 Load library from a file
- ✅ Input validation
- 🎯 Menu-driven console interface

```bash

```

2. Navigate to the project folder

```bash

```

3. Run the program

```bash
python main.py
```
